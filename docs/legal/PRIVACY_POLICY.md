# 彩加（Saika）プライバシーポリシー

最終更新日: 2026年6月7日

本プライバシーポリシーは、Discord 向けサーバー管理 Bot「彩加（Saika）」および付随する Web ダッシュボード（以下「本サービス」）における情報の取り扱いを定めるものです。本サービスを利用することで、本ポリシーに同意したものとみなします。

## 1. 運営者

本サービスは個人（以下「運営者」）が運営しています。

- 運営者: そのざき
- 連絡先: https://x.com/sonozaki_SZ

## 2. 取得する情報

本サービスは、機能の提供に必要な範囲で以下の情報を取得・保存します。

**Discord から取得する識別子**
- サーバー（ギルド）ID、チャンネル ID、ロール ID、ユーザー ID

**機能の利用に伴い保存する情報**
- ギルドごとの設定（言語、通知チャンネル、各機能の有効/無効 など）
- メンバーの参加・脱退イベント、参加経路、滞在期間（メンバーログ機能）
- メンバーの活動状況に関する情報（MemberActivity 等）
- チケット機能でやり取りされた内容
- リアクションロール・VC 関連・固定メッセージ等の設定
- メッセージ内容を扱う機能（固定メッセージ、フィルタ付き一括削除 等）で処理する範囲のメッセージ情報

**Web ダッシュボードの利用に伴う情報**
- Discord OAuth2 による認証情報（identify / guilds 等、付与されたスコープの範囲）およびセッション情報

**技術的に記録される情報**
- アクセスログ・エラーログ等（CDN / リバースプロキシを含む）

## 3. 利用目的

取得した情報は、本サービスの機能提供・維持・改善、不具合対応、および不正利用の防止のためにのみ利用します。

## 4. 保存場所・保存期間

- データは運営者が管理するサーバー上のデータベース（PostgreSQL）に保存されます。
- 保存期間は機能提供に必要な期間とします。**Bot をサーバーから退出させた場合、当該サーバーに関するデータは即時に削除されます。**

## 5. 第三者提供

運営者は、法令に基づく場合を除き、取得した情報を第三者に販売・提供しません。本サービスの提供にあたり、以下の外部サービスを利用します。

- Discord（本サービスの動作基盤）
- Cloudflare（通信の中継 / CDN）、VPS 提供事業者（サーバーホスティング）等

## 6. 利用者の権利

- サーバー管理者は、Bot をサーバーから退出させることで、当該サーバーに関するデータを削除できます（退出時に即時削除されます）。
- データの開示・削除をご希望の場合は、第 1 項の連絡先までご連絡ください。

## 7. セキュリティ

運営者は、取得した情報を適切に管理し、漏洩・改ざん・不正アクセスの防止に努めます。ただし、完全な安全性を保証するものではありません。

## 8. 年齢

本サービスは Discord の利用規約に従い、13 歳以上（お住まいの地域でより高い年齢が定められている場合はその年齢以上）の利用を前提とします。

## 9. 本ポリシーの変更

運営者は、必要に応じて本ポリシーを変更することがあります。重要な変更がある場合は、本サービスまたはサポートサーバー等で告知します。

## 10. 準拠法

本ポリシーは日本法に準拠します。
